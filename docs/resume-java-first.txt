三、项目经验
--------------------------------------------------------------------------------

智慧社区微服务管理平台
2025.09 - 至今 ｜ 独立开发
技术栈：Spring Boot + Spring Cloud Alibaba + MyBatis-Plus +
        MySQL + Redis + RabbitMQ + Docker

面向社区物业管理的微服务项目，涵盖物业缴费、活动报名、停车管理、工单处理等业务模块，
由我独立完成从数据库设计到接口开发的全流程。

1. 活动报名库存优化：活动报名场景下，高并发扣减库存导致数据库锁竞争和超卖问题。
   改为 Redis 预扣库存 + RabbitMQ 异步落库，将瞬时流量前移到缓存和消息队列层，
   超卖问题得到解决。

2. 停车计费幂等控制：停车场道闸上报频繁，重复抓拍存在重复计费风险。在计费接口中
   引入 Redis 分布式锁做幂等控制，同时调整加锁和事务开启的顺序，减少了高并发下
   数据库连接被长时间占用的情况。

3. 缓存设计与性能优化：对车位余量查询等高频接口使用 Spring Cache + Redis 做缓存，
   配合定时任务做缓存预热。压测中接口平均响应时间从 292ms 降至 3ms 以内，QPS
   有明显提升。

4. 系统稳定性保障：结合 Nacos 做配置动态管理，使用 Sentinel 在网关层做限流和服务
   降级。通过定时任务处理超时未支付订单，保证高并发场景下系统稳定运行。

5. AI 智能客服（探索实践）：基于 Spring AI 集成 DeepSeek，实现了关键字与向量语义
   结合的混合检索，将社区资料拼入 Prompt 生成回答。设计了无资料拒绝和回答校验机制，
   避免模型编造信息。通过这次实践，对 RAG 架构和大模型工程化落地有了基本理解。

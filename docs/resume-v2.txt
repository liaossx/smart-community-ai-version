三、项目经验
--------------------------------------------------------------------------------

智慧社区微服务管理平台
2025.09 - 至今 ｜ 独立开发
技术栈：Spring Boot + Spring Cloud Alibaba + MyBatis-Plus + Spring AI +
        MySQL + Redis + RabbitMQ + DeepSeek + Docker

面向社区物业管理的微服务项目，涵盖缴费、报名、停车、工单等业务模块，并在此基础上
引入大模型能力，构建了 AI 智能服务子模块。

1. RAG 智能客服：实现关键字 + 向量语义的混合检索，从知识库检索相关资料后拼入
   Prompt 交由 DeepSeek 生成回答。设计了加权合并算法（关键字主导 + 向量兜底），
   以及"无资料拒绝"机制防止模型编造。输出层通过 Normalizer 白名单校验 citation，
   确保每条引用可追溯。

2. 运营洞察与 Fallback：支持手动指标和自动 DB 聚合两种模式，模型基于报修、投诉、
   欠费等指标自动生成风险洞察和行动项。设计了 Fallback 降级机制：大模型不可用时
   自动切换为规则引擎保底输出，避免页面空白。

3. 工单智能分析：通过 System Prompt 约束模型输出结构化分类（故障类别、优先级、
   风险等级），并在输出层做优先级限幅校准 + 高危关键词（困人、起火等）二次判定，
   降低误判风险。

4. 活动报名库存优化：面对高并发下数据库锁竞争和超卖问题，改为 Redis 预扣库存 +
   RabbitMQ 异步落库，把瞬时流量扛在缓存和消息队列层，超卖问题得到解决。

5. 停车计费与缓存：通过 Redis 分布式锁实现计费接口的幂等控制，避免重复抓拍导致
   重复扣费。对高频查询接口做缓存预热，压测中平均响应从 292ms 降至 3ms 以内。

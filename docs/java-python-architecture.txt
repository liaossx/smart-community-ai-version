## 关键代码片段

### Java 调用 Python（AiEngineClient.java）

```java
@Component
public class AiEngineClient {
    private final RestTemplate restTemplate;

    // 同步模式：等 Python 返回完整结果
    public String analyze(String question, String context) {
        return restTemplate.postForObject(
            "http://python-engine:8000/analyze",
            new AiEngineRequest(question, context),
            String.class
        );
    }

    // 流式模式：接收 Python 的 SSE 流，逐步转发
    public Flux<String> analyzeStream(String question, String context) {
        return WebClient.create("http://python-engine:8000")
            .post()
            .uri("/analyze/stream")
            .bodyValue(new AiEngineRequest(question, context))
            .retrieve()
            .bodyToFlux(String.class);     // 每收到一个 chunk 就发射一次
    }
}
```

### Python 接收 Java 请求（ai_router.py）

```python
from fastapi import APIRouter
from pydantic import BaseModel

class AnalyzeRequest(BaseModel):
    question: str
    context: str

router = APIRouter()

@router.post("/analyze")
def analyze(req: AnalyzeRequest):
    # RAG 检索
    docs = rag_service.retrieve(req.question)
    # 拼 Prompt + 调 DeepSeek
    answer = llm_service.call(req.question, docs)
    return {"answer": answer, "sources": [d.source_id for d in docs]}
```
